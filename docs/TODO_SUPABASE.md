# ⚠️ IMPORTANT : Actions à réaliser dans Supabase

Avant de continuer, vous devez exécuter les scripts SQL dans votre dashboard Supabase.

## 🔴 ÉTAPE 1 : Créer les tables

1. Allez sur https://supabase.com/dashboard
2. Ouvrez votre projet **TeamEase** (anciennement `fo-metaux-dashboard`)
3. Cliquez sur **🗄️ SQL Editor** dans le menu de gauche
4. Cliquez sur **"+ New query"**
5. **Ouvrez le fichier** `SUPABASE_SCHEMA.sql` sur votre ordinateur
6. **Copiez tout le contenu** (Ctrl+A puis Ctrl+C)
7. **Collez dans Supabase** (Ctrl+V)
8. **Cliquez sur "Run"** (en bas à droite)
9. ✅ Attendez le message de succès

## 🔴 ÉTAPE 2 : Insérer les données

1. **Toujours dans SQL Editor**, cliquez sur **"+ New query"**
2. **Ouvrez le fichier** `SUPABASE_SEED.sql`
3. **Copiez tout le contenu** (Ctrl+A puis Ctrl+C)
4. **Collez dans Supabase** (Ctrl+V)
5. **Cliquez sur "Run"**
6. ✅ Attendez le message de succès

## 🔴 ÉTAPE 3 : Créer votre utilisateur

1. Dans Supabase, cliquez sur **🔐 Authentication** dans le menu
2. Cliquez sur **"Users"**
3. Cliquez sur **"Add user"** → **"Create new user"**
4. Remplissez :
   - **Email** : `votre.email@fo-metaux.fr` (ou tout autre email)
   - **Password** : Choisissez un mot de passe que vous retiendrez
   - ✅ **Cochez "Auto Confirm User"** (IMPORTANT !)
5. Cliquez sur **"Create user"**
6. ✅ Votre utilisateur est créé !
